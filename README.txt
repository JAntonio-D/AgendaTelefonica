/******* INTRUCCIONES DE INSTALACION **********/

1.- Abrir solución en VS.

2.- Abrir archivo Startup.cs, dentro de AgendaTelefonica

3.- Editar variable DB, poner la dirección correcta del archivo AgendaDB.mdf

4.- Compilar la solución

5.- Si marca errores, limpiar y recompilar

6.- Correr solución.

7.- Si al abrir la página sale un mensaje "Cannot GET /"
	
	Ir a carpeta ClientApp/src/app/Contactos y abrir Contactos.component.ts
	
	Modificar dicho archivo, poniendo un espacio o un salto de línea y guardar. Para que recompile

8.- Esperar a que se actualice la página o refrescar manualmente.
