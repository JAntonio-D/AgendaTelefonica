/******* INTRUCCIONES DE INSTALACION **********/

Para poder usar la Base de Datos es necesario contar con MS SQL Server 2017 LocalDB instalado.

1.- Abrir solución en VS.

2.- Compilar la solución

3.- Si marca errores, limpiar y recompilar

4.- Correr solución.

5.- Si al abrir la página sale un mensaje "Cannot GET /"
	
	Ir a carpeta ClientApp/src/app/Contactos y abrir Contactos.component.ts
	
	Modificar dicho archivo, poniendo un espacio o un salto de línea y guardar. Para que recompile

6.- Esperar a que se actualice la página o refrescar manualmente.
